# AASTP Data Structure

A structured representation of AASTP-1 Chapter 1 using JSON datasets, validation tools, and a future service layer architecture.

## Current Components

### Data

* interaction_rules.json
* distance_rules.json
* hazard_classes.json
* pes_types.json
* es_types.json

### Validators

* validateInteractions.js
* validateDistanceRules.js

## Objectives

* Provide machine-readable AASTP data
* Support national tailoring
* Support future APIs and service layers
* Support applications such as Excel add-ins and web calculators
